引言

终于还是受不了 Octopress 了，准备迁移到 Hugo，受不了 Octopress 是因为：

  • markdown code 语法支持不好，很多时候莫名其妙的 Build 失败，让人抓狂。（最主要原因）
  • 生成速度太慢，目前生成一次要2分多钟了
  • Octopress 项目已经没人维护了
